Тема та мета

Тема: Адаптивна верстка з використанням медіа-запитів.

Мета: Освоїти навички створення адаптивного дизайну під різні пристрої.

Опис реалізації

У даній роботі реалізовано адаптивну сітку за допомогою CSS Grid. Для десктопної версії застосовано три колонки (лівий блок, контент, правий блок), для планшетів — дві колонки, а для мобільної — одна.

Логічні оператори (наприклад, and, max-width) були використані для налаштування відповідної адаптації на різних екранах.
